About This Field Guide

This site is a personal wardrobe archive, part field guide, part museum catalog, part design collection. It exists to celebrate longevity, repair, craftsmanship, and intentional ownership, and to never, under any circumstance, function as a shopping app.

Every page reinforces one idea: clothing becomes more beautiful through wear and repair.

Philosophy

Built around the Indigo Americana Wardrobe Constitution, this collection favors natural fibers, workwear silhouettes, and museum-quality craftsmanship over fashion cycles. The goal is never to look fashionable. The goal is to look thoughtful.

On This Build

The site is static HTML, CSS, and JavaScript, no build step, no backend, no tracking, designed to be hosted directly on GitHub Pages. Wardrobe data lives in js/data.js as a plain JavaScript file, so the whole catalog can be edited directly, by hand, the same way a field notebook is.
